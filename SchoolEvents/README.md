# Информационна система за организиране и споделяне на училищни събития и инициативи

## Описание
Проектът представлява уеб приложение за управление и публикуване на училищни събития. 
Потребителите могат да разглеждат събития, а администраторите и учителите да създават и управляват съдържание.

## Технологии
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server / LocalDB
- Bootstrap
- JavaScript

## Основни функционалности
- Показване на предстоящи и минали събития
- Детайлна информация за събития
- Добавяне, редактиране и изтриване на събития
- Потребителски роли (Admin, Teacher)
- Календар на събития

## Стартиране на проекта
1. Отворете проекта във Visual Studio.
2. Проверете connection string в `appsettings.json`.
3. Отворете Package Manager Console.
4. Изпълнете:
   `Update-Database`
5. Стартирайте приложението.

## Забележка
При отваряне на проекта от GitHub се вижда само изходният код.  
Съдържанието (събития, потребители и др.) не се прехвърля автоматично, тъй като се съхранява в база данни.

Пълната работеща версия на проекта е предоставена отделно.
